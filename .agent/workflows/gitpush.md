# Git Push Workflow

// turbo-all

1. Stage all changes:
```
git add -A
```

2. Show what's staged:
```
git status
```

3. Commit with a descriptive message summarizing the changes:
```
git commit -m "<descriptive message based on staged changes>"
```

4. Push to remote:
```
git push
```
